## 🚀 How to Run

### Option 1: Local Development (XAMPP/WAMP)
```
1. Copy website folder to htdocs (XAMPP) or www (WAMP)
2. Start Apache server
3. Open: http://localhost/website/
```

### Option 2: Live Server
```
1. Upload website folder to public_html/
2. Ensure PHP 7.0+ is installed
3. Access: https://yourdomain.com/website/
```

### Option 3: Docker (Optional)
```
docker run -p 8000:80 -v ./website:/var/www/html php:7.4-apache
```
