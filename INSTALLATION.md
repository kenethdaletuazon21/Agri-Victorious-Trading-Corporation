### 🔧 To Run the Website

1. **Local Server (XAMPP/WAMP)**
   - Copy website folder to `htdocs` (XAMPP) or `www` (WAMP)
   - Start Apache
   - Navigate to: `http://localhost/website/`

2. **Live Server**
   - Upload website files to your hosting server
   - Ensure PHP 7.0+ is installed
   - Navigate to your domain: `https://yourdomain.com/website/`

3. **Image Setup**
   - Ensure logo.png, banner.png are in `website/images/` folder
   - Images are referenced in the pages
